# Tarte

Time Domain Acoustic Real-Time Engine is a project hosting simulation codes for various physical audio systems. Implementations are made in C++. Only the simulation core (engine) is in this repository: it is meant to be used as the basis of interfaces for e.g. plugins in other projects.

## Structure

The src folder contains the actual simulation codes.
