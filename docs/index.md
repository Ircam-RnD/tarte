# Welcome to tarte's documentation!

## Description

Tarte is a C++ library for real-time compatible simulation of various physical systems corresponding to (part of) musical instruments. The modeling relies on the port-Hamiltonian framework. Energy stable stepping-schemes are used, mostly based on the Scalar Auxiliary Variable (SAV) transformation.

Based on the library, real-time interfaces may be built as e.g. plugins of Max externals.
